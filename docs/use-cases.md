# Example Use Cases

Below are example scenarios illustrating how Accounting Anna can be used to automate and streamline financial data management.

---

## Use Case 1: Automated Bank Statement Aggregation
**Scenario:**
A user uploads monthly bank statements (PDF or CSV) from multiple accounts to OneDrive. Accounting Anna automatically extracts, cleans, and aggregates all transactions into a single master transaction stream for the year.

**Steps:**
1. User uploads statements to OneDrive.
2. Extraction and data preparation workflows run automatically.
3. Cleaned data is aggregated into a master stream.
4. User downloads the master transaction file for analysis or tax reporting.

---

## Use Case 2: Personal Accounting and Spending Analysis
**Scenario:**
A user wants to track and categorize personal spending across several accounts.

**Steps:**
1. User uploads statements to OneDrive.
2. Workflows process and aggregate the data.
3. Personal accounting workflow generates categorized spending reports and summaries.
4. User reviews monthly and yearly spending breakdowns.

---

## Use Case 3: Business Expense Management
**Scenario:**
A small business uses Accounting Anna to automate expense tracking and reporting for multiple employees and accounts.

**Steps:**
1. Employees upload statements to a shared OneDrive folder.
2. Workflows process, clean, and aggregate all transactions.
3. Business owner reviews consolidated expense reports and logs for compliance and budgeting.
